# Classes, Inheritance and Methods

At this point in the book, you should have established the difference between primitive types, create simple classes and methods and effectively use arrays, objects and control flow constructs. This chapter is a natural extension from the previous chapter where we combined methods and objects.

## Topic

This chapter explores the conventional object oriented programming concepts that is established in C#.

The chapter will explore the follow topic areas.

* Methods and Class Based Inheritance
* Properties and Property Methods
* Class Inheritance and Method Overriding
* Errors and Exceptions
* Encapsulation and Multiple Files/Libraries
* Namespaces and Code Organisation

We will be exploring more of the capabilities of classes and reusable components. In addition, this chapter will look into outlining some good conventions for organising your projects.
